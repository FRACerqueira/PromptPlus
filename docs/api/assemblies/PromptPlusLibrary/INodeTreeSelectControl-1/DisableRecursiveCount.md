### INodeTreeSelectControl&lt;T&gt;.DisableRecursiveCount method
</br>


#### Disables or enables recursive counting of child nodes in the selection control. Defauts to disabled.

```csharp
public INodeTreeSelectControl DisableRecursiveCount(bool value = true)
```

| parameter | description |
| --- | --- |
| value | A value indicating whether recursive counting should be disabled. If `true`, recursive counting is disabled; otherwise, it remains enabled. |

### Return Value

The current [`INodeTreeSelectControl`](../INodeTreeSelectControl-1.md) instance for chaining.

### Remarks

Disabling recursive counting may improve performance for large node trees by avoiding deep traversal when counting child nodes.
