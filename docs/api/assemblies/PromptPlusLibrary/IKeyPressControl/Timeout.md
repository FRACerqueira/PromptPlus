### IKeyPressControl.Timeout method (1 of 2)
</br>


#### Creates a key press control that waits for user input for the specified duration in milliseconds, returning a default key and modifiers if no input is received within the timeout period.

```csharp
public IKeyPressControl Timeout(int milliseconds, ConsoleKey defaultkey, 
    ConsoleModifiers? defaultmodifiers = default)
```

| parameter | description |
| --- | --- |
| milliseconds | The maximum amount of time to wait for a key press before returning the default key and modifiers. Must be a non-negative duration. |
| defaultkey | The key to return if the timeout elapses without user input. |
| defaultmodifiers | The modifier keys (such as Shift, Alt, or Control) to associate with the default key if the timeout elapses. If null, no modifiers are applied. |
