### IInputControl.IsSecret method
</br>


#### Masks the input as a secret, replacing characters with a specified mask character.

The following features cannot be used with `IsSecret`:

* SuggestionHandler
* DefaultEmpty
* Default
* HistoryEnabled

```csharp
public IInputControl IsSecret(char? value = null, bool enabledView = true)
```

| parameter | description |
| --- | --- |
| value | The character to use as the mask. Defaults is '#'. |
| enabledView | Enables the user to toggle the visibility of the masked input. Default value is true. |

### Return Value

The current [`IInputControl`](../IInputControl.md) instance for chaining.
