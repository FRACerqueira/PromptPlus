### IInputControl.MaxLength method
</br>


#### Sets the maximum length for the input and optionally the maximum width. The default vaue for *maxLength* is MaxValue characters.

```csharp
public IInputControl MaxLength(int maxLength, byte? maxWidth = null)
```

| parameter | description |
| --- | --- |
| maxLength | The maximum number of characters allowed for the input. |
| maxWidth | &gt;The maximum width of the input in characters.The value must be less than *maxLength* otherwise, it will be equal. |

### Return Value

The current [`IInputControl`](../IInputControl.md) instance for chaining.

### Exceptions

| exception | condition |
| --- | --- |
| ArgumentOutOfRangeException | Thrown if *maxLength* is less than 1. |
| ArgumentOutOfRangeException | Thrown if *maxWidth* is less than 1. |

### Remarks

The value of *maxWidth* must be less than *maxLength* otherwise, it will be set to equal.
