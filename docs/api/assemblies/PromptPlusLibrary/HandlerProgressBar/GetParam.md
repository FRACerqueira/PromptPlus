### HandlerProgressBar.GetParam&lt;T&gt; method
</br>


#### Retrieves the value of a context parameter identified by the specified key and casts it to the specified type.

```csharp
public T GetParam<T>(string? key = null)
```

| parameter | description |
| --- | --- |
| T | The type to which the parameter value will be cast. Must match the actual type of the stored parameter value. |
| key | The key that identifies the context parameter to retrieve. Cannot be null. |

### Return Value

The value of the context parameter associated with the specified key, cast to type *T*.

### Exceptions

| exception | condition |
| --- | --- |
| InvalidOperationException | Thrown if the context parameter collection is not available. |
| KeyNotFoundException | Thrown if a parameter with the specified key does not exist or cannot be cast to type *T*. |

### Remarks

Use this method to retrieve strongly typed context parameters. If the parameter exists but is not of the requested type, a KeyNotFoundException is thrown.
