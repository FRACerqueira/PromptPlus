### IConsole.RemoveCancelKeyPress method
</br>


#### Removes the current cancel key press (Ctrl+C/Break) handler and restores default behavior.

```csharp
public void RemoveCancelKeyPress()
```

### Remarks

This method remove handler previously set via [`CancelKeyPress`](./CancelKeyPress.md).

After calling this method:

* The [`BehaviorAfterCancelKeyPress`](./BehaviorAfterCancelKeyPress.md) setting will have no effect
* The [`UserPressKeyAborted`](./UserPressKeyAborted.md) setting to `false`.
