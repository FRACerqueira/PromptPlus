### IConsole.BehaviorAfterCancelKeyPress property
</br>


#### Gets the behavior to be applied after a cancel key (Ctrl+C/Ctrl+Break) is pressed. This setting is ignored if no handler is configured via [`CancelKeyPress`](./CancelKeyPress.md) or returns `false` from [`CancelKeyPress`](./CancelKeyPress.md).

```csharp
public AfterCancelKeyPress BehaviorAfterCancelKeyPress { get; }
```

### Property Value

A value from [`AfterCancelKeyPress`](../AfterCancelKeyPress.md) enum determining how the console should handle cancel events:

* Ignore - Continue running without aborting
* AbortCurrentControl - Abort only the current control's operation
