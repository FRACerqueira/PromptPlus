### ITableSelectControl&lt;T&gt;.AutoFill method
</br>


#### Automatically generates columns based on the public properties of the data class recognized by TypeCode.

```csharp
public ITableSelectControl AutoFill(int? minwidth, int? maxwidth = null)
```

| parameter | description |
| --- | --- |
| minwidth | The minimum width for auto-generated columns. When null, no minimum width is enforced. |
| maxwidth | The maximum width for auto-generated columns. When null, no maximum width is enforced. |

### Return Value

The current [`ITableSelectControl`](../ITableSelectControl-1.md) instance for method chaining.

### Exceptions

| exception | condition |
| --- | --- |
| InvalidOperationException | Thrown when AddColumn or AutoFit has already been configured. |
| ArgumentException | Thrown when *minwidth* and *maxwidth* contain invalid minimum/maximum width combinations. |

### Remarks

AutoFill cannot be used when AddColumn or AutoFit has already been configured. Properties with TypeCode.DBNull and TypeCode.Object are ignored. The column width is automatically adjusted based on the title size (property name) and the minwidth/maxwidth parameters, or content width when min/max width is null. Header alignment is always Center, and content alignment is always Left with sliding lines enabled.
