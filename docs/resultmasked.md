### Properties
[**Top**](#-promptplus--resultmasked)

- ObjectValue
	- Type : Object
	- ReadOnly	
	- Value with the result of the control.
	    - For generic type is string
	    - For date,time type is DateTime
	    - For Numeric,Currency type is double 

- Input 
	- Type : string
	- ReadOnly	
	- Value without mask.

- Masked 
	- Type : string
	- ReadOnly	
	- Value with mask.
